# 微信群聊机器人

_使用Python2.7_

基于wxpy开发的群聊机器人,可以在终端扫码登录微信后,获取好友或群聊列表,处理与回复收到的信息.目前核心功能是统计次数与定时触发提醒,在打卡群自用,属于高度定制,业务处理可能不适用于所有场景,可供学习参考,欢迎交流探讨.图灵聊天与定时提醒部分可以直接使用或修改自定义.

2018-09-19更新：自动同意好友添加请求
